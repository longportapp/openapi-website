# 行情命名词典

## TradeStatus - 交易状态

交易状态

| ID  | 描述            |
| --- | --------------- |
| 0   | 正常交易        |
| 1   | 停牌            |
| 2   | 退市            |
| 3   | 熔断            |
| 4   | 新股待上市      |
| 5   | 代码变更        |
| 6   | 待开盘          |
| 7   | 拆合股暂停交易  |
| 8   | 已到期 (衍生品) |
| 9   | 轮证待上市      |
| 10  | 终止交易        |

### Protobuf

```protobuf

  NORMAL = 0;
  HALTED = 1;
  DELISTED = 2;
  FUSE = 3;
  PREPARE_LIST = 4;
  CODE_MOVED = 5;
  TO_BE_OPENED = 6;
  SPLIT_STOCK_HALTS = 7;
  EXPIRED = 8;
  WARRANT_PREPARE_LIST = 9;
  SUSPEND_TRADE = 10;
}
```
