OpenAPI

程序化行情交易接口

通过统一接口获取港股、美股、A 股及新加坡市场的实时行情与历史数据，执行多种订单类型的交易操作。支持 HTTP REST 和 WebSocket 二进制长连接协议。

  • 完整市场覆盖 — 港股（窝轮、牛熊证）、美股（OPRA 期权）、A 股通、新加坡。L1/L2 深度、逐笔成交、资金流向。
  • 多层次交易 — 限价单、市价单、条件单、跟踪止盈止损。完整订单生命周期。
  • 6 种 SDK — Python、Node.js、Rust、Go、Java、C++。内置频率控制，开箱即用。
  • 历史数据 — 分钟到月线 K 线、财报、盈利预测、机构评级、内部人交易。

from longbridge.openapi import QuoteContext, Config # Initialize config = Config.from_env() ctx = QuoteContext(config) # Get real-time quotes resp = ctx.quote(["TSLA.US", "700.HK"]) for q in resp: print(f"{q.symbol}: {q.last_done}") # Subscribe to push ctx.subscribe(["TSLA.US"], SubType.Quote)
import { Config, QuoteContext } from 'longbridge' const config = Config.fromEnv() const ctx = await QuoteContext.open(config) // Get real-time quotes const quotes = await ctx.quote(["TSLA.US", "700.HK"]) quotes.forEach(q => { console.log(`${q.symbol}: ${q.lastDone}`) })
use longbridge::quote::QuoteContext; use longbridge::Config; let config = Config::from_env()?; let (ctx, _) = QuoteContext::try_new(config).await?; // Get real-time quotes let quotes = ctx.quote(["TSLA.US"]).await?; for q in quotes { println!("{}: {}", q.symbol, q.last_done); }
MCP

让 AI 助手直接调用行情与交易

托管的 HTTP MCP 服务。Claude Code、Cursor、Zed 等 AI 工具一行命令接入，OAuth 2.1 授权，无需配置 API 密钥。

  • 零密钥配置 — OAuth 2.1 一键浏览器授权，凭证自动刷新，随时可撤销。
  • 完整能力 — 实时行情、K 线历史、账户持仓、下单改单撤单。
  • 多客户端 — Claude Code · Cursor · Zed · ChatGPT · Codex · Cherry Studio。

CLI

AI-native 命令行全功能终端工具

覆盖全部 Longbridge 端点。行情查询、财报分析、技术诊断、订单管理、选股筛选——一条命令搞定。--format json 输出供 AI 代理消费。

  • 全功能行情 — 实时报价、L2 盘口、K 线、逐笔成交、资金流向、市场温度。
  • 深度财务 — 财报（利润表/资产负债表/现金流）、估值对标、EPS 预测、机构评级、内幕交易。
  • 跨市场选股 — 港美 A 股按市值+PE+技术信号筛选。MACD/KDJ/RSI 诊断。
