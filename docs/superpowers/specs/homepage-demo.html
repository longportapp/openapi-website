OpenAPI

程序化行情交易接口

通过统一接口获取港股、美股、A 股及新加坡市场的实时行情与历史数据，执行多种订单类型的交易操作。支持 HTTP REST 和 WebSocket 二进制长连接协议，满足从策略回测到实盘交易的完整需求。

  • 完整市场覆盖 — 港股（股票、ETF、窝轮、牛熊证）、美股（纽交所、纳斯达克、OPRA 期权）、A 股通及新加坡市场。支持 L1/L2 深度行情、逐笔成交、资金流向、市场温度指数。
  • 多层次交易功能 — 支持限价单、市价单、条件单（到价触发）、跟踪止盈止损等订单类型。完整的订单生命周期管理：下单、改单、撤单、查询。
  • 6 种语言 SDK — Python、Node.js、Rust、Go、Java、C++/C，均支持异步调用、Protobuf 序列化。SDK 内置频率控制与自动延迟机制，开箱即用。
  • 历史数据与财务分析 — 分钟到月线 K 线、历史行情回溯、财务报表、盈利预测、机构评级、内部人交易记录，支撑技术面与基本面研究。

from longbridge.openapi import QuoteContext, Config

# Initialize with environment variables
config = Config.from_env()
ctx = QuoteContext(config)

# Get real-time quotes
resp = ctx.quote(["TSLA.US", "700.HK", "AAPL.US"])
for q in resp:
  print(f"{q.symbol}: {q.last_done}")

# Subscribe to real-time push
ctx.subscribe(["TSLA.US"], SubType.Quote)
import { Config, QuoteContext } from 'longbridge'

// Initialize client
const config = Config.fromEnv()
const ctx = await QuoteContext.open(config)

// Get real-time quotes
const quotes = await ctx.quote(["TSLA.US", "700.HK"])
quotes.forEach(q => {
  console.log(`${q.symbol}: ${q.lastDone}`)
})
use longbridge::quote::QuoteContext;
use longbridge::Config;

let config = Config::from_env()?;
let (ctx, _) = QuoteContext::try_new(config).await?;

// Get real-time quotes
let quotes = ctx.quote(["TSLA.US", "700.HK"]).await?;
for q in quotes {
  println!("{}: {}", q.symbol, q.last_done);
}
MCP

让 AI 助手直接
调用行情与交易

托管的 HTTP MCP 服务，让 Claude Code、Cursor、Zed 等 AI 编程助手直接获取实时行情、管理账户、执行交易。采用标准 OAuth 2.1 授权流程，无需手动配置 API 密钥，浏览器一键授权即可开始使用。

  • 零密钥配置 — 标准 OAuth 2.1 授权流程，一键浏览器授权。凭证由 AI 客户端托管自动刷新，可在账户安全设置中随时撤销。
  • 完整行情与交易能力 — AI 可查询实时行情快照、K 线历史、账户资产与持仓，也可执行下单、改单、撤单操作（受账户权限与地区限制）。
  • 多客户端开箱即用 — 支持 Claude Code、Claude Desktop、Cursor、Codex、Zed、Cherry Studio 等主流 AI 工具。全球节点与中国大陆加速节点可选。

CLI

AI-native 命令行
全功能终端工具

覆盖全部 Longbridge 端点的命令行工具。实时行情查询、财报分析、技术面诊断、订单管理、持仓复盘——一条命令搞定。支持 --format json 结构化输出，AI 代理可直接调用。

  • 全功能行情 — 实时报价、L2 盘口深度、K 线（日线到分钟）、逐笔成交、资金流向、市场温度指数。覆盖股票、ETF、期权、窝轮。
  • 深度财务分析 — 完整财务报表（利润表、资产负债表、现金流量表）、估值指标对标（PE/PB/PS/股息率）、分析师 EPS 预测共识、机构评级、SEC Form 4 内幕交易记录。
  • 跨市场选股与技术诊断 — 港美 A 股叠加市值、PE、技术指标筛选。均线、MACD、KDJ、RSI、支撑压力位诊断，覆盖日线到 15 分钟级别。输出支持 Table、JSON、CSV、Markdown 格式。

SKILL

AI 助手的金融
能力扩展包

一条命令为 Claude、Cursor 等 AI 助手注入完整的 Longbridge 知识库。AI 自动识别股票代码和市场分析需求，智能调用行情查询、财务分析、选股诊断和交易执行。支持中英文混合提问。

  • 自动触发的智能识别 — 提到股票代码（AAPL.US、700.HK）、市场分析、持仓查询时自动激活。支持 /longbridge 前缀强制触发。
  • 全栈投资分析 — 跨市场选股（市值+PE+技术信号叠加）、技术面诊断（MACD/KDJ/RSI/支撑压力位）、财报对标分析、SEC 13F 机构持仓追踪、Form 4 内幕交易监控。
  • 完整知识参考 — 包含 CLI 命令全集、Python QuoteContext / TradeContext / ContentContext 三大上下文 API、Rust SDK 文档、全部类型定义与枚举值。
